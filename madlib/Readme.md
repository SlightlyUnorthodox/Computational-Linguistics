### Description
Adaptation of classic Mad Libs game in automated format...


### Requirements
This project requires the following to work properly:

* Python 2.7
* Packages
	* Tkinter
	* numpy
	* nltk

### Running Information

To run execute the command, **python madlib.py** , from terminal in directory containing repository.
